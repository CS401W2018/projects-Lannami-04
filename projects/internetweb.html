Internet

The Internet is a massive network that connects computers all around the world. It allows computers to communicate and share information to each other.

Web

The Web is a system of interlinked documents that we can access through the internet using web browsers like Firefox, Chrome, Edge and Safari. This is what we use to browse websites and was created to allow users to easily access and share information.

How are the Internet and Web different?

The Internet is like an underlying infrastructure that supports the Web. It consists of servers, routers, and cables that allow data to travel between computers. On the other hand, the Web is a service that runs on the Internet and allows people to access and share information.

ASCII and Unicode

ASCII (American Standard Code for Information Interchange) is a character encoding standard that represents text in computers. It assigns a unique code to represent a character. It allows 128 possible characters that include English letters, numbers, and basic punctuation. On the other hand, Unicode is like an expanded version of ASCII because it addresses the limitations set by ASCII. Unicode can accommodate different languages, symbols, and even emojis that ASCII cannot represent.

What is UTF-8

UTF-8 (8-bit Unicode Transformation Format) is a widely used encoding system for Unicode characters. It supports a wide range of characters from different languages. It uses one to four bytes to represent each character which makes it more efficient and flexible in handling complex characters.

URL

URL stands for Uniform Resource Locator. It is the address we type into our browser to access a specific website or page. It locates a specific page by knowing the protocol, domain, and path. For example, https://www.hello.com is a URL. These are essential for navigating content online.

Domain Name

The Domain name is the human-readable part of a URL. Like from our example earlier, hello.com is the Domain name which is much easier to recognize.

HTTP

HTTP (Hypertext Transfer Protocol) is the protocol that is utilized to send data across the Internet. It specifies the format and message transmission.

HTTPS

HTTPS (Hypertext Transfer Protocol Secure) is similar to HTTP but has more security. It makes the website safer to use by encrypting data transferred between your browser and the website.

IP Address

IP Address stands for Internet Protocol Address. It is a unique set of numbers assigned to each device on the Internet. Its purpose is to identify the device and locate it on the network. It is like a home address for our computer so other computers know where to send data.

DNS

DNS (Domain System Name) functions similarly to a phone book. DNS converts a domain name into a corresponding IP address so that our browser can connect to the appropriate web server and retrieve the desired content.

W3C Recommendations

These are guidelines and rules set by the World Wide Web Consortium (W3C) to make sure that web technologies complement one another and that it is consistent and accessible. They contribute to a more user-friendly Web experience.

XML, XHTML, HTML, HTML5

XML (Extensible Markup Language) is a way to structure data using tags. It is not specifically for web pages but for storing and transporting data. XHTML (Extensible Hypertext Markup Language) is a stricter version of HTML that follows XML rules. It is used to create web pages with more consistency. HTML (Hypertext Markup Language) is the standard language used to create web pages. It uses tags to define elements like headings, paragraphs, and links. HTML5 is the latest version of HTML. It includes new features and improvements, like better support for multimedia (videos and audio) and new elements for structuring web pages.
