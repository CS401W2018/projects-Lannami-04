1. What is JavaScript?

JavaScript is a programming language that adds interactivity and dynamic behavior to web pages, allowing users to perform actions like dragging, dropping, and updating content without reloading. It interacts with the Document Object Model (DOM) to manipulate elements directly on the page. This makes web pages more engaging and responsive to user input.

1.1 How did JavaScript get its name?

JavaScript got its name as part of a marketing strategy by Sun Microsystems to leverage the popularity of the Java programming language. Although JavaScript and Java are different, the name was chosen to attract more developers to web technologies. This branding helped boost JavaScript’s popularity on the web.

2. What are two ways you can run JavaScript?

JavaScript can be run in a web browser, where it interacts with the DOM to create interactive page elements. Another way to run JavaScript is through Node.js, which allows developers to execute JavaScript on a server, outside of a browser. This expands JavaScript's use beyond web development to server-side tasks.

3. What was the most popular front-end JavaScript library?

jQuery was once the most popular front-end library, simplifying tasks like HTML manipulation and event handling with cross-browser compatibility. It reduced the amount of code needed for common JavaScript tasks, making web development faster and easier. As a result, it became widely adopted by developers in its time.

4. What is the most popular front-end JavaScript library today?

Today, React.js is the most popular front-end library, known for its component-based structure and virtual DOM that improve performance. Developed by Facebook, React allows developers to build reusable UI components for complex applications. React Native extends React’s principles to mobile app development, further increasing its popularity.
