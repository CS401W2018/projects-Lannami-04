1. What is AJAX?

It is a web development technique that allows web pages to send and receive data from a server asynchronously without requiring a full page reload. Instead of refreshing the entire page, AJAX updates specific parts of a web page dynamically, making web applications faster and more interactive.

2. Why do you put your JS Code in a separate file?

We put our JavaScript code in a separate file to make it reusable across multiple pages. It also helps in maintaining the code easily and cleaner by separating the logic from the structure. It also improves performance as browsers can cache external JavaScript files and reduces time loads.
