2. What is the difference between an ID and a Class?

- ID: Described as a noun, it uniquely identifies one specific element on the page, using #
- Class: Referred to as an adjective, it can be applied to multiple elements to describe their style or behavior. Itâ€™s reusable and denoted by a period

3. What is the difference between a SPAN tag and a DIV tag?

- SPAN: An inline element used for styling or grouping smaller parts of text or inline elements.
- DIV: A block-level element used to structure larger portions of a webpage.

4. What is the difference between a DIV tag and a HEADER, MAIN, FOOTER?

- DIV: A general-purpose block container with no specific meaning, used for layout purposes.
- HEADER: An element for introductory or navigational content.
- MAIN: Represents the main content of the webpage.
- FOOTER: Contains footer information, such as copyright or link.
