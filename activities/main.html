1. Why is the viewport meta tag important?

This tag is important because it controls the layout of a webpage by defining how it should be scaled and displayed on different screens. This ensures that our site is user-friendly and accessible to all.

2. How can you adjust the CSS for printing?

Use a separate print stylesheet. You can add specific rules in the current styling of the webpage by using media print. By using this, we can change things like font sizes or hide parts that we do not want to show in the printout.

3. When would you want to adjust the CSS for printing?

We want to adjust the CSS for printing when we want the printed pages to look nice and easy to read. For example in our class earlier, we removed the navigation links in the printout because we won't be needing them. Some other eamples include making texts bigger or changing the layout so it fits nicely on the page when someone prints it out.
