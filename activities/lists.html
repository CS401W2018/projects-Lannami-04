  3. What are special entity characters?

    4. Special entity characters are specific symbols used to represent characters that already has a reserved meaning in HTML. These are used to ensure that symbols like <,>, &, and others are correctly displayed on a webpage rather than being interpreted as part of the HTML code.

  4. What is the difference between OL and UL Tags?

    5. The difference of OL and UL tags lies in how they display lists. OL tags create an ordered list where each item is numbered or lettered sequentially. The default styling is a numbered list but you can also use letters or Roman numerals. On the other hand, UL tags creates an unordered list. Each item is marked with a bullet point as the items do not have an inherent order. You can also change the style from circles to squares.
