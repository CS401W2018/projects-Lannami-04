1. Why don't we use the border property?

We dont use the border property on tables because CSS can't change the default border styling of the table itself. Our professor emphasized that we shouldn't rely on HTML attributes to control presentation, instead, we use CSS for styling. By using CSS, we gain greater flexibility and control over how elements appear on the page.

2. What is the easiest way to color every other row a different color?

The easiest way to color every other row in a table is to use the nth-child pseudo-class in CSS.

3. When should we use a table?

We should use tables when we need to present tabular data which are information organized into rows and columns. For example, we can use it to present data like reports, schedules, or comparisons.
